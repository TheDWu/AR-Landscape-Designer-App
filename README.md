# AR-Landscape-Designer-App
Design and visualize the backyard of your dreams! <br>
This app enables you to visualize your backyard in 3D, making it easier to plan and design your dream outdoor space

## Features
- Preview products in real-time 3D <br>
- Multiplayer: Up to 100 users can conncurently join the same session and edit/design the space <br>
- Cost Estimator: Every object that is placed in the 3D space will be added to running total

## Tech Stack 
- Vuforia
- Normcore
- Unity UI
- AR Core
- Sketchfab
